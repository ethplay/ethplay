Ethplay
========
 .. image:: ethplay_small.png
     :width: 200px
     :alt: Ethplay logo
     :align: center

Ethplay is a platform for implementation and management of gaming applications on the Ethereum blockchain.

Ethplay provides basis functionalities for managing interactions between game developers, game hosts, players, investors, advertisers, and more.
Using Ethplay, hosting a game is as easy as sending a transaction.

Ethplay aims to become a Distributed Autonomous Organization (DAO), completely owned and managed by its token holders.

Core Functionalities
---------------------
* Several game types are built into the platform and are ready to be used by anyone.
* Game developers can create new game types. Game development is divided into backend and frontend development tasks. Both are incentivized.
* Users can host games. The host of a game is the owner of that instance. Games can be hosted without special programming skills (sendig a transaction).
* Investors can invest in games types and/or instances and receive profits.
* Ethplay is fueled by a token with enhanced functionalities: dividend distribution, platform governance, and in-game currency.
* 100% of platform profits are transferred to token holders.
* Platform governance is conducted by token holders: decisions on platform development, game inclusion/promotion, and network parameters.

Supported Game Types
----------------------

Casino games
++++++++++++
* Several casino games are implemented, and are ready to be used by game hosts.
* Other game types can be implemented by game developers.
* A central smart contract (House) acts as the hub for managing different game types, game instances, players, and game-user interactions.
* Ethplay provides secure random number generation to all game instances that request it.
